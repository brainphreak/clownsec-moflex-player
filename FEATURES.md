# CLOWNSEC VIDEO — Complete Feature List

## Playback

* Native moflex/MobiClip video decoding
* 2D & 3D auto-detection
* Plays Moflex videos inside `.cia` files
* Responsive draggable seeking
* Resume playback saved per video
* Watched / unwatched badges
* Volume boost to 400%
* Locking screen to avoid accidental touch
* Bottom screen toggles off during playback
* HQ color toggle (Old 3DS performance boost)

## Subtitles

* External SubRip (`.srt`) subtitles
* Auto-load subtitles with movie name, or load any SRT
* Subtitle settings for depth, location, language and delay saved per movie

## Library

* Library browser by Genre, Category, and built-in search
* Poster Art, Year, Description, Size, Runtime, 3D/2D etc.
* Scrape movie information from online catalog
* Manually update metadata for custom genres and info
* Hide or show system files and non-movie files for cleanliness

## Downloads

* Catalog browser to download direct from Clownsec, Zackk's, and the 3DS Movie archive
* Fast downloads optimized to reach up to 5 Mbps (625 KB/s) on New 3DS
* Download queue for automating multiple downloads
* Resumable transfers for continuing download later
* TV Season zips extract themselves
* Direct URL download
* Download auto-retry on failure

## Battery & Power

* Downloads continue when lid is closed to save power
* Notification LED when download completes and lid is closed
* Choose to sleep or power off 3DS when download queue is finished
* Idle screen-off after 5 minutes without activity

## Extras

* Built-in Wi-Fi file server to upload files from PC, tablet or phone
* File manager to browse, rename, move and delete files
* Default themes and custom color picker for custom user themes
* Recently played list for quick access
* Automatic update checks for new releases
